My own work

A website for an existing real estate agency

Front-end

CSS, JavaScript, Semantic UI

Back-end

EJS templates, Node.js, Express, MySQL

Source code: GitHub

This bilingual website displays the offers of a real estate agency: 8 random ones on the homepage, offers by category, or a chosen real estate with its details. The CRUD operations allow the admins to add, edit, update or delete the offers. Images are uploaded through Cloudinary's widget. New user receives a greeting e-mail after registering. The option of password reset is available.

Source code: GitHub

Back-end

Node.js, MongoDB, GraphQL
