My own work

A website for an existing real estate agency

Front-end

CSS, JavaScript

Back-end

EJS templates, Node.js, Express, MySQL

Source code: GitHub

This bilingual website displays the offers of a real estate agency: 8 random on the homepage, offers by category, or a chosen real estate with its details. The CRUD operations allow the admins to add, edit, update and delete the offers. Images are uploaded through Cloudinary's widget. New user receives a greeting e-mail after registering. The option of password reset is available.

Source code: GitHub

Back-end

Node.js, MongoDB, GraphQL

Source code: GitHub

The top 10 scores of players (signed in users) get displayed on the Leaderboard. The user who beats the highest score gets a flash notification. All of the user's scores and levels achieved are available on the My Scores page.
The game itself was created based on Thomas Weibenfalk's YouTube tutorial video.
